¿Qué acciones predefinidas incluir?

Selecciona todas las que quieras. La consola PowerShell libre va incluida siempre.

⟳

Reiniciar PC

shutdown /r /t 30 — da 30s al usuario para guardar

⊟

Apagar PC

shutdown /s /t 30

🔒

Bloquear pantalla

Bloquea sesión activa del usuario (rundll32 user32 LockWorkStation)

⚙

Reiniciar servicio

Campo para escribir nombre del servicio → net stop / net start

✕

Matar proceso

Campo para nombre/PID → taskkill /F /IM nombre.exe

🗑

Limpiar temporales

Borra %TEMP% y C:\Windows\Temp — sin confirmación

≡

Ver procesos activos

Muestra lista de procesos en la consola (tasklist)

💾

Estado del disco

wmic logicaldisk — muestra uso por unidad
